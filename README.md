# ⚙️ Marmara Mekatronik - C Programlama Yolculuğu

Bu depo, bir **Mekatronik Mühendisliği** öğrencisinin bilmesi gereken temel ve ileri seviye C programlama konularını içeren şahsi çalışma notlarım ve kod örneklerimden oluşmaktadır.

## 🚀 İçerik ve Kapsam

Projeler, donanım seviyesine yakın programlama mantığını anlamak ve mekatronik sistemlerdeki algoritma karmaşıklığını yönetmek üzerine kurgulanmıştır.

### 🏗️ İşlenen Temel Konular
* **Diziler ve Fonksiyonlar:** Veri gruplama ve modüler kod yazımı.
* **Pointer (İşaretçi) Mantığı:** Bellek yönetimi ve doğrudan adresleme.
* **Sensör Veri Filtreleme:** Pointer ve dizileri kullanarak hatalı verilerin (gürültü) temizlenmesi.
* **Kalibrasyon Algoritmaları:** Fırın sıcaklık kontrolü gibi gerçek dünya senaryoları.

## 🛠️ Kullanılan Araçlar
* **Donanım:** HP Victus
* **Derleyici:** GCC (GNU Compiler Collection)
* **Terminal:** Git Bash & zsh
